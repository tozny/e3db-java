Class ClientBuilder

  • Object
    • com.tozny.e3db.ClientBuilder

  • public class ClientBuilder
extends Object
    Use this class to configure and create an E3DB client.

    Two shortcut methods are available for creating a client:

    • fromConfig(Config) — Creates a client based on a Config object.
    • fromCredentials(ClientCredentials, String) — Creates a client based on credentials and a private key.

    Otherwise, the following methods must be called before build will succeed:

    • setApiKey(String)
    • setApiSecret(String)
    • setClientId(UUID)
    • setPrivateKey(String)
