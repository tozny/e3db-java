com.tozny.e3db

Interface ClientCredentials


  • public interface ClientCredentials
    Holds information necessary to authenticate with E3DB. See Client.register for registering and capturing credentials; Use Config.fromJson(String) and Config.json() to convert credentials to a standard JSON format.
