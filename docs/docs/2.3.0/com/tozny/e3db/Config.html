com.tozny.e3db

Class Config

  • Object
    • com.tozny.e3db.Config

  • public class Config
extends Object
    Holds all necessary information to create an E3DB client.

    Use this class to save and load credentials and other information about a given client. The json() method will return a String containing a JSON document that can be later used to re-create the Config object using the fromJson(String) static method.

    Once a Config instance has been created, use the ClientBuilder.fromConfig(Config) method to configure a ClientBuilder that can re-create the client.

    The fromCredentials method(s) can also be used create a Config instance, if you do not want to use the built-in JSON document.
