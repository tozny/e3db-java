Class OutgoingSharingPolicy

  • Object
    • com.tozny.e3db.OutgoingSharingPolicy

  • public class OutgoingSharingPolicy
extends Object
    Describes records that a client shares.

    This class describes a sharing relationship from the writers's perspective (that is, "who am I sharing with?").

    Use Client.getOutgoingSharing(ResultHandler) to get a list of all record types shared by a client.
