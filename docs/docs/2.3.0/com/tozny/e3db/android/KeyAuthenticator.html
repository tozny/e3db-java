Class KeyAuthenticator

  • Object
    • com.tozny.e3db.android.KeyAuthenticator

  • public abstract class KeyAuthenticator
extends Object
    Provides an interface for authenticating with the user, by gathering a password, asking for a fingerprint, or by entering a device PIN. The authentication method used is determined by the value returned by the KeyAuthentication.authenticationType() method for the KeyAuthentication instance associated with the configuration stored.

    When authentication is needed, this class will call the appropriate method (getPasssword, etc.) with a handler argument. The implementation should gather the appropriate authentication, and then call the handle method on the handler that was given.

    Use the static method defaultAuthenticator(Activity, String) to get an authenticator implementation that can gather a password, collect a fingerprint, or ask the user to enter their lock screen PIN.

    The static method noAuthentication() can be used to get an authenticator that should not be called (useful when the configuration is not protected by any special authentication).
